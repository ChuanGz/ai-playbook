# Support

## Questions and corrections

Search existing GitHub issues before opening a new one. Use an issue to report unclear, contradictory, outdated, or unusable guidance, and include the affected section, decision context, and supporting evidence.

This project does not provide implementation consulting, system certification, guaranteed response times, or private support. Questions about a specific vendor, application, or organization policy should be handled by the responsible project or organization.

## Security and conduct

Do not report suspected vulnerabilities or sensitive information in public issues. Follow [SECURITY.md](SECURITY.md). Report conduct concerns using the private method in [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
