# Changelog

All notable user-visible changes to this project are documented here. The project follows semantic versioning for documentation contracts as defined in [MAINTENANCE.md](MAINTENANCE.md).

## 1.0.0 - 2026-07-04

### Added

- Durable OSS repository governance and warning-free documentation CI.
- Long-lived documentation architecture, maintenance policy, and maturity model.
- AI system principles, solution-selection guide, decision-record template, and glossary.
- Adoption, AI-assisted delivery, context, retrieval, knowledge, tools, MCP, memory, orchestration, and agent-runtime contracts.
- Evaluation, release, security, governance, observability, operations, cost, degradation, and incident guidance.
- End-to-end workflow guide and bounded RAG, MCP tool, and agent examples.
- Reader lifecycle and capability/evidence diagrams.

### Boundaries

- No vendor implementation, application framework, compliance certification, or production-readiness claim is included.
- Consuming projects remain responsible for policy, thresholds, implementation, validation, and operating evidence.
