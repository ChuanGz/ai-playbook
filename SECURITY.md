# Security Policy

## Reporting a vulnerability

Do not disclose suspected vulnerabilities in public issues or discussions. Report them privately through [GitHub private vulnerability reporting](https://github.com/ChuanGz/ai-playbook/security/advisories/new) or email `thanhnhan.lionus@gmail.com`.

Relevant reports include unsafe workflow behavior, dependency or action compromise, malicious links or examples, accidental disclosure, and guidance that could create a concrete security failure when followed as written.

Include the affected file or version, impact, reproduction steps, proof of concept when safe, and suggested mitigation. Allow the maintainer time to investigate before public disclosure. No response or remediation timeline is promised.
